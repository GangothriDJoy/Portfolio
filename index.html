SevaASHA
Integrated Work Management System for ASHA Workers: a cross-platform mobile app coordinating maternal health monitoring across five healthcare stakeholder groups.
Problem solved
ASHA workers and healthcare stakeholders lacked a unified, role-aware system to track maternal risk and immunization schedules in real time.
Key features
Role-based access control (RBAC) workflows for 5 stakeholder groups; real-time dashboards and alert routing via Firestore listeners for maternal risks and emergencies; automated immunization tracker for upcoming, completed and overdue vaccinations.
Challenges & learnings
Keeping the Firestore listeners fast and reliable as the number of stakeholder roles grew was the hardest part; it pushed me to think carefully about RBAC-based data structuring instead of just bolting permissions on afterward.
